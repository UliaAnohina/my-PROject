# my-PROject
Этот репозиторий содержит финальный проект курса по автоматизации тестирования. В нем реализованы автоматизированные тесты веб-приложения с использованием Python, Selenium WebDriver и паттерна Page Object для обеспечения читаемости и поддерживаемости кода. Проект также демонстрирует соблюдение основных принципов Code Style (PEP 8) и ООП.
